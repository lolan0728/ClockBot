## ClockBot GitHub Pages Assets

This folder is ready to publish with GitHub Pages.

Use it like this:

1. Push this repository to GitHub.
2. Open the repository's `Settings` -> `Pages`.
3. Under `Build and deployment`, choose `Deploy from a branch`.
4. Select branch `main` and folder `/docs`.
5. Wait for Pages to publish.

After publishing, the icon URL will look like:

```text
https://YOUR-USERNAME.github.io/YOUR-REPOSITORY/clockbot-icon.png
```

Open that image URL on your iPhone first. If Safari can load it, Bark can usually use it as the `icon` URL too.
